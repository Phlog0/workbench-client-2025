# Workbench Client

**Интерактивный конструктор электрических схем для объектов электроэнергетики (35 кВ / 10(6) кВ / 0.4 кВ).**

Workbench Client — это мощный инструмент для инженеров-проектировщиков, позволяющий создавать однолинейные схемы в визуальном редакторе с использованием технологий Drag-n-Drop и баз знаний типовых решений.

## Ключевые возможности

- 🖱️ **Интерактивное построение схем**
  Полностью визуальный интерфейс. Просто перетаскивайте элементы (трансформаторы, выключатели, разъединители) на рабочую область.
- 🗄️ **База типовых решений**
  Используйте готовые блоки и узлы из встроенной библиотеки, чтобы не рисовать однотипные элементы с нуля.
- 🤖 **Умные подсказки (Инженерный AI)**
  Система анализирует действия пользователя и предлагает корректные с точки зрения ПУЭ варианты продолжения схемы.
- 💾 **Импорт/Экспорт в JSON**
  Сохраняйте проект в структурированном формате и обменивайтесь им с коллегами. Загрузите JSON-файл обратно в редактор, чтобы продолжить работу с того же места.
- 🖼️ **Работа с подложками**
  Загрузите на рабочую область существующий чертёж или эскиз (JPG/PNG), чтобы использовать его как подложку для обводки или сверки с оригиналом.
- 📄 **Экспорт в PDF (Alpha)**
  Генерируйте чертеж вашей схемы в формате PDF для печати или передачи заказчику. _(Функция находится в стадии активного тестирования)_
- **Поддержка классов напряжения:**
  - Высокое напряжение: 35 кВ
  - Среднее напряжение: 10(6) кВ
  - Низкое напряжение: 0.4 кВ
